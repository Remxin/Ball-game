# 🎮⚽ **Ball Game** ⚽🎮
## Classic internet game in a form of **web application**.
## 📱 Works also on mobile devices!


### Launch the application on local machine:  
1. Enter project directory
2. Launch localserver 
3. Enjoy the game 😊


### 🤖 **Technologies**
- [x] TypeScript
- [x] Webpack
